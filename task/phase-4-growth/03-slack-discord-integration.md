# Task: Slack / Discord Integration

**Phase:** 4 - Growth (Post-Launch)
**Status:** pending

## Description

Build integrations with Slack and Discord for team notifications and task management from chat.

### Features

- Notifications for task events (created, assigned, status changed, commented)
- Slash commands for quick task operations from chat
- Thread-based task discussion sync

### Deliverables

- Slack app/bot implementation
- Discord bot implementation
- Notification configuration (per-project, per-user)
- Slash command handlers
- Setup documentation for both platforms

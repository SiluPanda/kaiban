# Task: Agent Activity Review UI

**Phase:** 3 - Ecosystem (Weeks 9-12)
**Status:** pending

## Description

Build UI views dedicated to reviewing what AI agents have done across the project.

### Views

- **Agent activity feed**: Timeline of agent actions across the project. Shows what agents did, when, and to which tasks.
- **Session review**: Review a specific agent's work session -- tasks touched, changes made, comments left.

### Requirements

- Filter by agent, time range, and action type
- Clear attribution of human vs. agent actions
- All AI-generated content visually marked as such
- Drill-down from activity feed to specific task detail

### Deliverables

- Agent activity feed page
- Session review page
- Agent-specific filtering
- Activity timeline component
